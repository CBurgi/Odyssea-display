# Odyssea Display

## Start Program
### Terminal 1 (Front End)
1. cd into Odyssea-display/frontend
2. if not installed, run `npm i` to install node_modules
3. run `npm start`

### Terminal 2 (Back End)
1. cd into Odyssea-display/backend
2. if not installed, run `npm i` to install node_modules
3. run `npm start`

Once running, the site can be seen on `localhost:3000`

## Test Program
### Terminal 3 (Test)
1. cd into Odyssea-display/test_script
2. if not in venv
    a. run `python3 -m venv venv`
    b. run `source venv/bin/activate`
3. if not installed, run `pip install -r requirements.txt`
4. run `python3 push.py`
